对焦

1. 拍不清楚的几种可能性

1.1 快门速度太慢
：造成相机抖动，一旦抖动，那么没有任何物体是清楚的

1.2 被摄物体移动

1.3 天气不太好

雾气使画面呈现“低对比度”（低对比度画面不是模糊），在雾气的情况下，近处的物体不会影响，远处的物体发白发灰

1.4 天气越恶劣，照片越精彩

平流雾（从上往下拍）、云雾（从下往上拍）

1.5 曝光过度、不足
：曝光过度、不足时使画面中主体太亮或太暗而失去细节

1.6 冷凝水
：镜头上凝结的雾气

如果外出低温环境拍摄，回去之后，相机像眼镜一样，出现雾气，冷凝水，
这时候就需要注意了，如果冷凝水过多，会导致相机短路，这时候我们就需要去掉电池，等待相机回温。
当然最好的办法是当我们要回去的时候，把相机让进包里面，让包从外到里慢慢回温，这样就不会出现冷凝水的现象。
雨天或者游泳拍摄，使用专业的相机雨衣或潜水罩。

1.7 焦外虚化
：前景虚化、背景虚化（不会抢焦点物体的风头）

1.8 对焦失误
：对焦到背景上、对焦到空气中、对焦没在眼睛上（离相机近的那一只眼睛）

2. 焦点，聚焦之处

2.1 焦点是什么呢？

内容的“焦点”：
Focal Point = 重点、核心、关键点

光学上的“焦点”
Focal Plane = 焦平面、焦点平面

    2.1.1 焦平面：所有位于焦平面上的物体都会是完全清晰的（前、后焦平面）

2.2 对焦：照片清晰的前提

光学中心到前焦平面的距离与到后焦平面的距离之比是一个固定的值
对焦：移动镜头中的玻璃片使前焦平面的光汇聚在后焦平面上

3. 对焦目标的选择

3.1 对焦命令

3.2 对焦点
：对焦的目标区域

多点对焦：相机自动选择对焦点——“就近”原则。
        何时适用：
        - 使用特殊视角拍摄
        - 来不及瞄准拍摄对象
        何时不适用：
        - 有前景遮挡的时候
单点对焦：“人”指定一个对焦点
        如何控制对焦点的位置：拨轮

  • 误区一：对焦点≠焦点

    对焦点是在屏幕上对对焦区域的选择、焦点是以距离而论的

  • 误区二：多点对焦（启用全部对焦点）= 画面全部清晰？（×）

    虽然启用了全部对焦点，但是只有离相机最近的焦平面会被拍摄清楚

4. 自动对焦模式

4.1 单次对焦
：AF-S = Auto Focus - Single
/ 自动对焦 —— 单点

何时适用：风光、静物、静止的人物
何时不适用：距离无规则变化的物体：球类运动摄影、宠物摄影 / 手持拍摄微距
4.2 连续对焦
AF-C = Auto Focus - Continue
/ 自动对焦 —— 连续
旋转相机，保证对焦点始终位于主角之上
何时适用：可预测运动规律的事物
何时不适用：不可预测运动规律的快速移动物体

4.3 跟踪对焦
：根据运动速度“猜测”焦点距离变化、根据颜色判断焦点位置变化、根据亮度判断焦点位置变化

何时适用：体育类摄影、快速运动的人（小朋友、舞台摄影）
何时不适用：焦距目标太小、对焦目标与背景颜色、亮度太接近

4.4 人脸优先对焦
：相机会识别人脸（几乎不受前景的影响）

何时不适用：前景中有人脸出现；人物太小，脸被忽略

对焦点 + 对焦模式
单点对焦 + 单次对焦 / + 连续对焦 （将焦点始终放在连续对焦的物体上）/ 无法 + 跟踪对焦
多点对焦 + 单次对焦（就近原则） / + 连续对焦（其本质就是跟踪对焦）

5. 手动对焦

自动对焦失败的时候使用手动对焦

5.1 自动对焦的原理

反差式对焦：反差式对焦的工作原理和手动对焦的工作原理是完全一样的（多次调整镜组内镜片尝试找到一个最佳的位置）通过画面的反差或者对比度来判断画面是否清晰，对比度越高说明画面越清晰。由于反差对焦衡量的是对比度，所以对比度明显（反差大）的场景更适用于反差对焦

相位式对焦：相位对焦元件（相机底部），一字对焦点，横向一字对焦点，十字对焦点

缺点：1. 对光的要求高，光线不足影响对焦 2. 镜头之间存在差异，特别是副厂镜头，经过复杂计算得到的镜片移动距离会出现误差，所以可能会出现轻微对焦不准，或者失焦的情况（需要微调）

5.2 自动对焦失败的原因

太近：距离太近不能对焦，有一个“最近对焦距离”
无反差：不要将相机对准毫无结构的物体进行对焦
结构方向：竖着的一字对焦点不能对竖行结构（例如竖柱子）容易失败，横向一字对焦点拍摄横向物体容易失败
昏暗：
被阻挡：解决办法：中央单点单次对焦 / 手动对焦

5.3 何时使用手动对焦模式

  • 完全的前景遮挡
  • 弱光环境
  • 反差极小的场景
  • 手动对焦镜头（不支持自动对焦）

一、焦距

1. 什么是焦距

焦距的定义：光学中心与光线汇聚点之间的距离称之为焦距

2. 焦距与可视角度

可视角度

    焦距越大，前方可视角度越狭窄，CMOS上的图像就越大。
    PS：全画幅相机与非全画幅相机的区别：
    全画幅相机相较于非全画幅相机画幅较大，且可视角度大于非全画幅相机的可视角度，
    所以全画幅相机拍照非常开阔，非全画幅相机拍照把物体拉的比较大

3. 等效焦距

等效焦距：非全画幅相机的某一焦距“等于”全画幅相机哪种焦距的视觉效果？这种换算称之为“等”“效”焦距

    全画幅： * 1
    APS-C画幅： * 1.5
    佳能APS-C画幅： * 1.6
    M4/3画幅： * 2

例如 M4/3画幅相机使用与全画幅相机一样的镜头（300mm焦距），300mm焦距的镜头使用在M4/3画幅相机上等效于全画幅使用600mm的镜头

疑问：“非全画幅镜头，能用在全画幅机身上吗？”

—— 投影到COMS上的光不足以覆盖全画幅COMS，所以不行

4. 焦距的分类

  • 标准镜头：50mm
  • 中焦镜头：28mm - 105mm
  • 广角镜头：＜ 28mm；超广角镜头：＜ 20mm；准广角镜头：35mm
  • 长焦镜头：＞ 105mm；中长焦镜头：＞ 85mm；超长焦镜头：＞ 300mm

5. 定焦和变焦

  • 定焦镜头的优势

      1、轻便    2、成像好    3、光圈大

二、 焦距对立体透视的影响

立体透视及空间感

立体透视

    1. 透视消失点：一个三维立体物体会有三个消失点，他们的边相互不平行，边的延长线永远都会相交在这三个消失点上
    2. 不同焦距的立体透视效果是不同的：
                  - 广角镜头：三个消失点很近，平行线极为不平行，它们（透视辅助线）汇聚地更急促，立方体更扭曲（近处的物体变得很大，远处物体变得非常小）
                  - 中焦镜头： 汇聚变地不是那么急促了，相当于人眼的感觉，“近大远小”没有广角镜头那么明显（更像是讲故事，栩栩如生）
                  - 长焦镜头：汇聚地更接近于平行线，“近大远小”的效果变得不是那么明显了

空间感

    - 广角镜头1st：空间感十足
    - 中焦镜头2nd：空间感薄弱了一些
    - 长焦镜头3rd：几乎没有了距离的概念，就像是一幅画儿

三、 不同焦距的特点及运用

1. 广角镜头的效果

明显的近大远小

  1. 一秒钟瘦脸法
  2. 无中生有的大小对比（突出我们的主体，本来是一样的大小，生生被我们凸显出主角）
  3. 夸张原有的大小比例
  4. 颠倒大小的比例

强烈的视觉冲击力

  1. 线条的汇聚
  2. 极端的大小对比
  3. 夸张的距离感
  4. 跳的更高

放大狭小的空间

容纳更多的景物

2. 中焦镜头的效果

最真实的透视感

像人眼一般记录当时的场景

朴实的叙述方式

用生活中的大白话来讲故事，中焦镜头太过于朴实，所以来表达故事难度很大

适当的沟通距离

与被摄者之间的距离是恰当的，相比于广角镜头需要离被摄主体太近，相比于长焦镜头需要离被摄主体又太远，不好沟通

不多不少的画面元素

超大的光圈

终于可以在暗处环境拍照了，可以保证快门速度在中速以上

明显的焦外虚化

不要盲目的去做虚化效果

3. 长焦镜头的效果

平面化的视觉效果

没有近大远小的立体透视效果，所以像一幅画儿

猎奇感十足

极端的简化

画面元素非常少，长焦视角非常狭窄

极端的焦外虚化

4. 错位摄影

一种幽默的摄影方式 – 使用小光圈来拍摄（需要前景、背景都很清楚）

5. 特种镜头

人像镜头
：大光圈的定焦镜头 35mm（全身照、大半身照片+环境一起）、50mm（极大光圈、半身像）

柔焦镜头
：让画面朦胧起来

微距镜头
：拍摄现场，影像的大小大于物体；

鱼眼镜头
：极端的广角，180°的可视角度的广角镜头

移轴镜头
：消除了仰拍时的立体透视（近大远小）

增距镜
：上在机身和镜头之间

附加镜头
：附加镜头安装在镜头之前

显微镜
：科研摄影

系统性地学习拍摄

摄影基础知识

摄影基础知识分为七部分：曝光三要素、常用的操作模式、测光、白平衡、曝光补偿、景深以及存储格式

1. 曝光三要素

  • 光圈:它是相机控制进光量大小：光圈开的越大，进光量就越多。光圈通常用字母“F”来表示，“F”值越大，光圈越小。
  • 快门：它是控制相机进光量的快慢，在同样的光圈和感光度下，快门速度时间越长，那么进光量就越多，照片就会越亮。
  • 感光度（ISO）：它是控制照片明暗的，它与光圈和快门的原理不同，快门和光圈都是控制进光量的，而感光度是提升相机感光元件灵敏度的。简单来说：感光度越低，照片越暗；感光度越高，照片越亮。随着感光度的提升，照片的噪点也会越多，画质就会有所下降。

2. 常用的操作模式

  • P档（Program Exposure）：程序自动档，戏称为“傻瓜模式”
  • Av档（Aperture value：Aperture Priority Auto Exposure）：光圈优先档，在此模式下，调节光圈的大小，在拍摄时，快门时间根据现场环境自动调节，保证拍出曝光正常的照片。Av档的优势在于在拍摄时仅需要设置好光圈大小来控制景深，无需考虑快门速度。
  • Tv档（Time value：Shutter-Priority Auto Exposure）：快门优先模式，在此模式下，调节快门速度的快慢，在拍摄时，光圈大小根据现场环境自动调节，以保证拍出曝光正常的照片。Tv档比较适合拍摄运动的物体，尤其是快速奔跑的物体。
  • M档（Manual）：手动模式，在这种模式下，光圈、快门、感光度都需要自己根据现场情况手动调节。

3. 测光

  • 平均测光：它是将取景画面分成不同的几块、分别对各部分进行测光，最后取平均值，相机根据这个平均值来进行曝光。
  • 点测光：它是只对取景画面中的一小部分进行测光，相机会根据这一小块区域的亮度进行测光后对照片进行曝光。点测光比较适用于光线明暗对比大的场景，比如日出、日落。
  • 中央重点测光：它是对取景画面的中央60%，四周40%进行测光，这种测光模式可以保证你拍出的照片不会过亮或是过暗。
  • 局部测光：它是以取景画面某一局部进行测光，主要是以取景框中心的一个圆圈中的光线来进行测光。

4. 白平衡

白平衡

5. 曝光补偿

在拍摄照片遇到过曝或者欠曝时，可以利用曝光补偿来进行修正。曝光补偿向左调，照片会变暗，反之。

6. 景深

景深指照片中能够清晰成像的最近的物体与最远的物体之间的部分。景深主要与光圈和焦距有关“光圈越大，景深越小，反之越大；焦距越大，景深越小，反之越大

7. 存储格式

  • jpeg格式：压缩的存储格式，大部分的设备可以读取此格式，如若不需要特别多的后期工作，使用此格式足够。
  • raw格式：数码相机原始存储格式，能够储存更多的数据，对于后期的处理提供了巨大的便利，但是文件大小更大。
